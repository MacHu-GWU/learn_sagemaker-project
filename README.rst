Install
------------------------------------------------------------------------------

``learn_sagemaker`` is released on PyPI, so all you need is:

.. code-block:: console

    $ pip install learn_sagemaker

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade learn_sagemaker
